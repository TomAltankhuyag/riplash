## Command Line Instructions
### Frontend
1. `cd frontend\`  
2. `npm run start`

### Backend
#### Script Start
1. `cd backend\`
2. `bash start.sh`
### #Manual Start
Optionally, each process can be manually started in the appropriate directories.

1. `cd backend\load_balancer\`
2. `npm run start`

Using another terminal:

2. `cd backend\process\`
3. `npm run start`
